## 通用约束（默认底线）

- **ARCH-001** 上层依赖下层，禁止反向依赖（Controller → Service → Repository / Domain）
- **ARCH-002** 同层模块之间禁止循环依赖
- **ARCH-003** 跨服务调用必须经过 RPC / HTTP 客户端封装层，禁止业务代码直接构造网络调用
- **ARCH-004** 领域实体禁止直接对外暴露，对外契约统一通过 DTO / VO 表达
- **ARCH-005** 第三方 SDK 必须在适配层包装，禁止业务代码直接耦合 SDK 类型
