## 通用规则（默认底线）

- **AMT-001** 金额计算必须使用 `BigDecimal`，禁止 `double` / `float`
- **AMT-002** 金额存储统一使用 `BIGINT`（单位：分）或 `DECIMAL(18, 2)`，禁用 `FLOAT` / `DOUBLE`
- **AMT-003** 加减乘除必须显式指定 scale 与 RoundingMode，禁止使用默认 scale
- **AMT-004** 货币单位必须随金额字段一同存储与传递，禁止假定单一币种
- **AMT-005** 对账聚合金额必须按业务唯一键去重，避免重复计入
